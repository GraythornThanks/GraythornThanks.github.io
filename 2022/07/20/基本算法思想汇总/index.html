基本算法思想汇总

位运算

bit是度量信息的单位，包含0和1两种状态。熟悉掌握位运算，能够帮助我们理解程序运行中的种种表现，提高程序运行的时空效率，降低编程复杂度

1. 基本位运算

与 或 非 异或
& | ~ ^

2. 移位运算

算术左移 算术右移
<< >>

算术右移等于“除以二向下取整”，而除以二在C++中实现为“除以二向0取整”

例：快速幂

每一个正整数可以唯一表示为若干指数不重复的2的次幂的和。

如果b在二进制下有k位，其中第i位的数字是 c1c_1c1​ ，那么

b=ck−1∗2k−1+ck−2∗2k−2+⋯+c0∗20{ b = c_{k-1}*2^{k-1} + c_{k-2}*2^{k-2} + \cdots + c_0*2^0 }b=ck−1​∗2k−1+ck−2​∗2k−2+⋯+c0​∗20

于是：

ab=ack−1∗2k−1∗ack−1∗2k−1∗⋯∗ac0∗20{ a^b = a^{c_{k - 1} * 2 ^ {k - 1}} * a^{c_{k - 1} * 2^{k - 1}} * \cdots * a^{c_0 * 2^0} }ab=ack−1​∗2k−1∗ack−1​∗2k−1∗⋯∗ac0​∗20

我们可以通过k次递推求出每个乘积项，当c_i = 1时，把该乘积项累积到答案中。b & 1可以取出 b 在二进制下的最低位，而b >> 1运算可以舍去最低位，在递推的过程中将二者结合，就可以遍历b在二进制表示下的所有位数c_i。

整个算法的时间复杂度为O(log_2 b)

3. 二进制状态压缩

将一个长度为m的bool数组用一个m位二进制整数表示并存储

操作 运算
取出第k位 (n >> k) & 1
取出后k位 n & ((1 << k) - 1)
第k位取反 n ^ (1 << k)
第k位赋值1 n | (1 << k)
第k位赋值0 n & (~(1 << k))

例：最短Hamilton路径

给定一张 n(n <= 20) 个点的带权无向图，点从 0 ~ n - 1 标号，求起点 0 到终点 n - 1 的最短Hamilton路径

Hamilton路径的定义是从0到n - 1不重不漏地经过每个点恰好一次

事件复杂度为 O(n2∗2n)O(n^2*2^n)O(n2∗2n)

4. 成对变换

对于非负整数n:

当n为偶数时，n ^ 1等于n + 1
当n为基数时，n ^ 1等于n - 1

这一性质常用于图论邻接表中边集的存储，将一条无向边分别存储在n与n + 1的位置(n为偶数)

5. lowbit运算

lowbit(n) = n & (-n)

求出最低位的1及后面所有的0,配合hash可以找出二进制下所有是1的位

前缀和与查分

1. 前缀和

作用：数列a某个下标区间的和，可表示为前缀和相减的形式

S[i] = a[1] + a[2] + ... a[i];
a[l] + ... + a[r] = S[r] - S[l - 1];

构造：

for (int i = 1; i <= n; i ++ ) s[i] = s[i - 1] + a[i];

2. 差分

作用：将对原序列的区间操作转化为差分序列上的单点操作

b[1] = a[1];
b[i] = a[i] - a[i - 1];

构造：

b[l] += c, b[r + 1] -= c; //给区间[l, r]中的每个数加上c

二分

二分查找的平均时间复杂度和最坏时间复杂度均为 O(log⁡2n)O(\log_2n)O(log2​n)

1. 整数二分

在单调递增序列a中查找 >=x>=x>=x 的数中最小的一个（x或x的后继）

在单调递增序列a中查找 <=x<=x<=x 的数中最大的一个（x或x的前驱）

C++ STL中

lower_bound() 查找第一个 >=num>=num>=num 的数字
upper_bound() 查找第一个 >num>num>num 的数字

2. 实数二分

确定好所需要的精度eps，在mid上判定选择即可

双指针

双指针，就是同时使用两个指针，在序列、链表结构上指向的是位置，在树、图结构中指向的是节点，通过或同向移动，或相向移动来维护、统计信息

1. 维护区间信息

双指针维护区间信息的最简单模式就是维护具有一定单调性，新增和删去一个元素都很方便处理的信息

例：乘积小于 K 的子数组

给定一个长度为 n 的正整数数组 nums 和整数 k, 找出该数组内乘积小于 k 的连续子数组的个数。

2. 子序列匹配

例：判断子序列

给定一个长度为 n 的整数序列 a1,a2,…,ana_1,a_2,…,a_na1​,a2​,…,an​ 以及一个长度为 m 的整数序列 b1,b2,…,bmb_1,b_2,…,b_mb1​,b2​,…,bm​
请你判断 a 序列是否为 b 序列的子序列。
子序列指序列的一部分项按原有次序排列而得的序列，例如序列 {a1,a3,a5} 是序列 {a1,a2,a3,a4,a5} 的一个子序列。
